How to Import a ZIP File into Eclipse

Follow these steps to import a ZIP file containing an Eclipse project:

  1. Download the ZIP File (containing a project) to your computer but do not place in the eclipse workspace folder: Click here to donwload ZIP
  2. Open Eclipse: You do not need to create a new project because the ZIP file contains an existing project.
  3. Import the Project:
    1. Click on File in the menu bar and select Import....
    2. In the Import Wizard, expand the General category and select Existing Projects into Workspace. Click Next.
    3. Select the Select archive file option and click the Browse... button
    4. Navigate to the location where you saved the ZIP file, select it, and click Open
    5. The wizard will display the project(s) contained in the ZIP file. Make sure the desired project is checked
    6. Check if the box Copy projects into workspace is selected
    7. Click Finish to complete the import process
  4. Verify the Imported Project: The project should now appear in your Package Explorer. Double-click it to open and review its contents.

Remember: a HashMap<String, ArrayList<Integer>> means each city name is linked to a ArrayList of temperatures.

You know how to add values to ArrayList. The only difference is you need to get to the values in HashMap first:

hashmap.get(city); // you get to the value

Then you add your temperature reading to that list:

hashmap.get(city).add(temp);

This works because map.get(city) returns the list of temperatures for that city, and then .add(temp) appends the new reading onto that list.
