Box & Pointers Reference Guide

Be on the lookout for making these common errors:

  • Primitive variable has an arrow and is pointing to a rectangle (this is incorrect)
  • Forget to update primitive's value box with new value, e.g., increment
  • Object variable does not have a pointer box and corresponding rectangle
  • Arrow points in wrong direction, i.e., pointing from rectangle to pointer box
  • Object's variable box points to two different rectangles
  • Object's variable box points to incorrect rectangle
  • Failed to update to where arrow should point
  • No arrow pointing to null when object variable is referencing null
  • String object is drawn as if it is a primitive type
  • Class name is missing at the top of the Object's rectangle
  • Array not drawn correctly
