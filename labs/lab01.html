üí° Hint: New Line in printf

You can add a new line in two ways when using printf:

  • \n ‚Üí works on most systems, commonly seen in examples.
  • %n ‚Üí recommended, because it automatically uses the correct line ending for the operating system (Windows, Mac, Linux).

Both of these will print two lines:


System.out.printf("Shop: %s\nAverage Price: $%.2f\n", shopName, averagePrice);
System.out.printf("Shop: %s%nAverage Price: $%.2f%n", shopName, averagePrice);

üí° Hint 1: Start simple

Try writing the first few numbers by hand: 1, 11, 111, 1111. How do you get from one to the next mathematically?

üí° Hint 2: Growing the number

If you have 11 and want 111, think:
11 * 10 = 110, then add 1 ‚Üí 111.

Each time through the loop, the current value is multiplied by 10 * to shift its digits left, and then 1 is added to append a new 1 at the end. * For instance, if current is 11, multiplying by 10 produces 110, and adding 1 * results in 111.

üí° Hint 3: The update rule

The pattern each time is:
current = current * 10 + 1;
