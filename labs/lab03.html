💡 Hint: sumAll

This is a good place for nested loops:

  1. Make a variable like sum and start it at 0.
  2. Use the outer loop to go through each row.
  3. Use the inner loop to go through each value in that row.
  4. Add each value to sum.
  5. Return sum at the end.
💡 Hint: sumRow

You only need one loop here:

  1. Pick the row using grid[rowIndex].
  2. Loop through every value in that row.
  3. Add each value to a running total.
  4. Return the total.
💡 Hint: countValue

This is another nested loop pattern:

  1. Start with count = 0.
  2. Go through every row and every column.
  3. Check whether grid[r][c] == target.
  4. If yes, increase count by 1.
  5. Return count.

💡 Hint: countOccurrences

Think of the map as storing each word and how many times it appears:

  1. Create a new HashMap<String, Integer>.
  2. Loop through each item in the array.
  3. If the item is not in the map yet, put it in with value 1.
  4. If it is already there, update its count using map.get(item) + 1.
  5. Return the map at the end.
💡 Hint: getScoreOrDefault

This is a safe lookup pattern:

  1. Check whether the map contains the key using containsKey(name).
  2. If it does, return scores.get(name).
  3. If not, return 0.
💡 Hint: totalScore

You do not need the keys here, only the values:

  1. Start with sum = 0.
  2. Loop through scores.values().
  3. Add each value to sum.
  4. Return sum.
📚 Quick Reference: 2D Arrays & HashMaps

2D Array Operations

Operation Syntax Description
Number of rows grid.length Total number of rows in the 2D array.
Number of columns in a row grid[r].length Length of row r; rows may be different sizes.
Access an element grid[r][c] Value at row r, column c.
Loop over all rows for (int r = 0; r < grid.length; r++) Outer loop moves through each row.
Loop over columns in a row for (int c = 0; c < grid[r].length; c++) Inner loop moves through columns.
Nested loop pattern grid[r][c] Use inside for loops to read values.

HashMap Operations

Operation Syntax Description
Add or update a key/value pair map.put(key, value) Inserts or replaces the value for key.
Get value for a key map.get(key) Returns the value or null if key doesn't exist.
Check if key exists map.containsKey(key) Returns true if key is in the map.
Remove a key/value pair map.remove(key) Deletes entry for key if present.
Get all keys map.keySet() Returns a Set of all keys in the map.
Get all values map.values() Returns a Collection of all stored values.
Check size map.size() Returns number of key/value pairs in the map.
Increment a count safely map.put(k, map.get(k) + 1) Common pattern for building frequency maps.
