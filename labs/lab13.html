Step 4. Create a new method: public void update() that moves horizontally and clamps the object to the screen.

Clamping a simple operation in programming and mathematics to restrict a value so that it never falls outside a specified range. If a value is below the minimum bound, you get the minimum; if it is above the maximum, you get the maximum; otherwise, you keep the original value. Math.max(min, Math.min(value, max));

  1. Increments x by speed. Note you need to cast speed to int 
  2. // Clamp x within [0, screenWidth] 
x = Math.max(0, Math.min(screenWidth, x));
