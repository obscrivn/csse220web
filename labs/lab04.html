Create a class called DataTypePractice that includes a main method and a custom method to print the data type of a given value. The method is called printDataType that takes an Object as an argument.

Important Note: When you pass a primitive value (like 5) to a method that expects an Object, it is automatically converted (autoboxed) to its corresponding wrapper type (e.g., Integer). No need for extra code like ((Object)int).

public static void printDataType(Object value){ //TODO finish this method }

This method should print a statement in the following format for primitives:

The value [value] is a primitive and the data type is [TypeName].

This method should print a statement in the following format for objects:

The value is an object and the data type is [TypeName]

Example: If the input is 5, the output should be:

The value 5 is a primitive and the data type is Integer.

Write Javadocs

Include proper Javadoc comments for the printDataType method (place it above the method declaration).

Test Your Code: In the main method, call your printDataType method with various values:
  • An int literal
  • A double literal
  • A char literal
  • A String
  • An object Array
  • An object ArrayList

Note you must initialize arrays first before you pass to the method

HINT 1: For ArrayList - consider accessing an item of the array. If you just access the class of ArrayList, you will only get ArrayList as a data type

HINT 2: instanceof is only used with classes. But if the primitive is converted to an Object, you can use a Wrapper Class.

printDataType(5) // Expected: The value 5 is a primitive and the data type is Integer. printDataType(arr) // Expected: The value is an object and the data type is int[].

printDataType(string) // Expected: The value is an object and the data type is String.
