Lab: Exploring Polymorphism, Upcasting & Downcasting

In this lab you will work with a simple class hierarchy to see how Java chooses which print() method to call at runtime (polymorphism), how upcasts let you treat a subclass as its superclass, and how downcasts let you recover subclass‚Äêspecific behavior, sometimes safely and sometimes resulting in errors if misused

1. Setup & Code

Copy the following into PolymorphismDemo.java


public class PolymorphismDemo {
    public static void main(String[] args) {
        A x = new D();
        x.print();               // (1)
        D x2 = (D)x;
        x2.print();              // (2)
        x2.printC();             // (3)
        B x3 = (B)x;
        // x3.printC();         // (4) 
    }
}

class A {
    public void print() { System.out.println("A"); }
}

class B extends A {
    @Override public void print() { System.out.println("B"); }
}

class C extends B {
    @Override public void print() { System.out.println("C"); }
    public void printC() { print(); }
}

class D extends C {
    @Override public void print() { System.out.println("D"); }
}

Note: removing print() in D will cause Java to invoke the next override up the chain, demonstrating dynamic dispatch

2. Tasks

  1. Predict the output: Without running the program, write down what lines (1), (2), and (3) print.
  2. Verify your prediction: Run java PolymorphismDemo and confirm the actual output
  3. Test removing D.print(): Comment out the @Override public void print() in D, recompile, and rerun. Explain which version of print() is invoked and why.
  4. Compile-time vs. runtime error: Uncomment the line x3.printC();, attempt to compile, and observe the error. Then change it to: 
    
if (x3 instanceof C) {
    ((C)x3).printC();
}
    and show that it now compiles and runs without exception
  5. Illegal downcast: Change the first line to A x = new B(); and keep the cast D x2 = (D)x;. Run the code and observe the ClassCastException. Explain why this happens

3. Reflection Questions

  • What is upcasting? Why does A x = new D(); require no explicit cast?
  • What is downcasting? Why must you write (D)x when moving from A back to D?
  • Dynamic dispatch: How does the JVM decide which print() to call at runtime?
