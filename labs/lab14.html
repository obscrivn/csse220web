Adding ActionListener to the Button

Now that you have your layout ready, let‚Äôs make the button do something! Whenever the ‚ÄúCount‚Äù button is clicked, it should call the increment() method that updates the number in the text field.

  1. Inside the constructor (after adding your components), register the listener for the button using a lambda expression: 
    button.addActionListener(e -> increment());

    The expression e -> increment() means: ‚ÄúWhen the button is clicked, run the increment() method.‚Äù

  2. Implement the increment() method in your class. It should:
    1. Increase the counter value by 1
    2. Update the text field to display the new count
    private void increment() {
    count++;
    field.setText(Integer.toString(count));
}

    üí° Make sure your count and text are declared as instance variables (outside the constructor), so they can be accessed inside increment(). 

    private int count = 0;
private JTextField text; // declared at class level

üí° Hint: Why Not Declare count Inside increment()?

If you declare int count = 0; inside the increment() method, it will reset to zero each time you click the button. Using an instance variable ensures the value is remembered between clicks.

üí° Hint: One-Line vs Multi-Line Lambdas

A one-line lambda (like e -> increment()) does not need braces or semicolons. If you want to perform multiple actions, use braces and semicolons inside: 

button.addActionListener(e -> {
    count++;
    field.setText(Integer.toString(count));
});
