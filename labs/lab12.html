My Counter

In this lab, you will create a Swing component with a button that increments the count

Task Description

  • A JLabel to describe the counter
  • A non‑editable JTextField to display the current count
  • A JButton labeled “Count” that, when clicked, increments the counter

Adding ActionListener to the Button

Whenever the “Count” button is clicked, we want to execute setValues() method:

  1. Add the listener registration at the end of the constructor: 
    button.addActionListener(e -> setValues());
  2. Implement the setValues() method to:
    1. Increment the counter
    2. Update the text field: text.setText(count + "");

My Accumulator

Overview

In this lab you will create a Swing JPanel that reads integer values, keeps a running total, and displays the accumulated sum. You already have a viewer class with a JFrame set up, you will build just the panel class.

Task Description

1. Setup

  1. Create a new class named MyAccumulator that extends JPanel
  2. Ensure your project has necessary Swing imports
  3. Add your panel to the existing frame in the viewer’s main() method

2. Layout

Use a 2×2 grid to arrange the labels and text fields. For example: 

setLayout(new GridLayout(2, 2, 5, 5));

– Row 1: A label “Enter an Integer:” and an editable text field for input
– Row 2: A label “Accumulated Sum:” and a non‐editable text field for output

Note: because yoru grid is 2x2, the panel will display your components in the order (see the screenshot) when you use add()

  • Reads the integer from the input field. Note to read input use getText(). It is a good practice to trim text from spaces .getText().trim()
  • Adds it to a running total variable (you need Integer here)
  • Clears the input field
  • Updates the output field with the new sum (you need String here)

                to parse Integer to string use Integer.toString()

Use a lambda expression to register the listener (no loops or error checks needed here).
