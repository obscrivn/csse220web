Rotating a Face

In this lab, you will work with a Swing component that draws a face and applies rotation. You are provided with a starting file (RotateComponent.java)

Task 1: Experiment with Rotation

In the provided code, there is a variable degreesToRotate that controls the angle of rotation. Modify this value (for example, try 0, 30, 60, or 90 degrees) to observe how the rotation affects the drawn face.

Task 2: Rotate Around the Face's Center

Currently, the face rotates around its upper-left corner. Modify the code so that the rotation occurs around the center of the face.

  • Hint: To rotate around the center, adjust the drawing coordinates.

Undoing Transformations

After applying translations and rotations, it is important to undo these transformations to avoid affecting subsequent drawing operations.

Step 3: Refactoring Your Code

After you have successfully applied rotation your face drawing, refactor your code to improve its structure and readability by breaking it into methods similar to the above section of the lab.

  1. void drawCenteredFace(Graphics2D g2d, int faceLocX, int faceLocY, int faceWidth, int faceHeight) – Draws the rectangle representing the face
  2. drawLeftEye(Graphics2D g2d, int faceLocX, int faceLocY) – Draws the left eye on g2d w.r.t. faceLocX and faceLocY
  3. drawRightEye(Graphics2D g2d, int faceLocX, int faceLocY) – Draws the right eye on g2d w.r.t. faceLocX and faceLocY
  4. drawNose(Graphics2D g2d, int faceLocX, int faceLocY) – Draws the nose w.r.t. faceLocX and faceLocY

Recursion with Triangular Shapes

In this lab, you will practice recursion by computing the area of a triangular shape composed of stacked unit squares. The triangle is formed like this: 

  []
  [][]
  [][][]
  [][][][]

Objective

Your task is to implement a Triangle class that calculates its area using recursion. The area is defined as the sum of the number of squares in each row.

Instructions

  1. Create the Triangle Class:
    • Define a private instance variable width (an integer) representing the number of rows in the triangle.
    • Implement a constructor that accepts an integer parameter to initialize width.
  2. Implement a Recursive Method:
    • Create a method (for example, computeAreaRecursively) that takes an integer parameter (representing the current row or width) and returns the area computed recursively.
    • This method should include:
      1. A base case to stop recursion (e.g., if the input width is less than 1, return 0).
      2. A recursive case that adds the current width to the area computed for width - 1.
  3. Test Your Implementation:
    • Add a main method in the same file as your Triangle class
    • Create one or more instances of Triangle with various widths
    • Print the computed area for each instance to verify that your recursive method works correctly
