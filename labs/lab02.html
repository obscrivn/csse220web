Exercise 1: Sum of an Array

Resources 1: Arrays basics

Implement the method sumArray that takes an array of integers and returns the sum of all its elements.

Hint: Use a loop to iterate over the array and accumulate the sum in a variable.

Exercise 2: Maximum Value in an Array

Implement the method maxArray that finds and returns the maximum value in a given array of integers.

Hint: Initialize a variable with the first element of the array. Then loop through the array and update the variable whenever a larger value is found.

Exercise 3: Reverse an Array

Implement the method reverseArray that returns a new array with the elements in reverse order compared to the input array.

Hint: Create a new array of the same length as the input array, then use a loop to copy elements from the end of the input array to the beginning of the new array.

Exercise 4: Count Occurrences in an Array

Implement the method countOccurrences that returns the number of times a specified integer appears in an array.

Hint: Loop through the array and increment a counter each time you encounter the target value.
