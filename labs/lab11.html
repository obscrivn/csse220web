Instructions

Step 1: Centering the Face with Translation

You are provided with a TranslateComponent.java file that draws a face (a rectangle for the face, two eyes, and a nose). However, the face is currently drawn with its upper-left corner at (0, 0).

Your Task: Modify the paintComponent(Graphics g) method to use a translate command so that the face always appears centered in the window. Use this.getWidth() and this.getHeight() to compute the correct offsets.

Step 2: Refactoring Your Code

After you have successfully centered your face drawing, refactor your code to improve its structure and readability by breaking it into methods. This makes your code modular and easier to maintain.

Create the following methods:

  1. drawFace(Graphics2D g2d) – Draws the rectangle representing the face
  2. drawLeftEye(Graphics2D g2d) – Draws the left eye
  3. drawRightEye(Graphics2D g2d) – Draws the right eye
  4. drawNose(Graphics2D g2d) – Draws the nose

Encapsulation

These methods are declared as private and void:

  • Encapsulation: They are internal to the class and not intended to be accessed from outside. This keeps the public interface of your class clean.
  • Implementation Hiding: Declaring these methods as private prevents other classes from relying on these internal details. This means you can change the implementation later without affecting other parts of your program.

Passing the Graphics Object and Local Variable Initialization

You only pass the Graphics2D object to each method. This object is initialized in the paintComponent method and contains the current state (like transformations) needed for drawing.

Inside each method, you declare and initialize the local variables specific to that drawing task. For example, in the method to draw the right eye, the coordinates and dimensions are defined locally: 

    // Draw right eye	 Original Implementation	
    int rightEyeLocX = 130;
    int rightEyeLocY = 20;
    int eyeWidth = 50;
    int eyeHeight = 50;
    g2d.drawRect(rightEyeLocX, rightEyeLocY, eyeWidth, eyeHeight);

   // Refactored Implementation
  private void drawRightEye(Graphics2D g2d) {
      int rightEyeLocX = 130;
      int rightEyeLocY = 20;
      int eyeWidth = 50;
      int eyeHeight = 50;
      g2d.drawRect(rightEyeLocX, rightEyeLocY, eyeWidth, eyeHeight);
  }

This design ensures that each method is self-contained, handling its own drawing logic using the shared Graphics2D context.
