Lab Assignment 05: Creating a Class

Resources: Understanding Fields and Classes

In this lab, you will create a class called StudentGradebook that represents a student's record of assignments. You will also use the provided StudentGradebookMain file to test your implementation.

The main focus of this lab is to learn about:

  • Creating class instances (objects) using constructors.
  • Implementing methods that perform actions (e.g., adding assignments, printing reports).
  • Interacting with objects from a client program (the main method).

Instructions

  1. Import a Main Java File : Download java file.
  2. Create a New Class: In Eclipse, create a new Java class named StudentGradebook in an appropriate package.
  3. Implement Constructors:
    • Create a default constructor that sets the student's name to "Studenty McStudenton" and initializes an empty grade list.
    • Create a parameterized constructor that accepts a String for the studentâ€™s name and initializes an empty grade list.
  4. Add fields: You need a field for student's name and a field that can store students names and grades
  5. Implement Methods:
    • addAssignment(String assignmentName, double grade): Adds an assignment and its grade to the student's grade record.
    • printGradeReport(): Prints the student's name along with each assignment and its grade.
    • Submit your StudentGradebook Class Java file
  6. Use the Provided Main File:

    Open the provided file StudentGradebookMain.java. Uncomment the example code inside the main method to test your StudentGradebook class.

    The main method should create two instances of StudentGradebook (one using the parameterized constructor and one using the default constructor), add several assignments to each, and print out grade reports.

  7. Extra Credit (Optional):

    If you finish early, add a new method to compute and print the student's average grade as part of the grade report.

Hint: Follow the example in StudentGradebookMain.java carefully. Make sure that the assignment names and grades are stored properly so that printGradeReport() can display them. Also, pay attention to the default constructor's expected name.
