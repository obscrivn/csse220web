Lab Assignment 05: Creating a Class

Resources: Understanding Fields and Classes

In this lab, you will create a class called StudentGradebook that represents a student's record of assignments. You will also use the provided StudentGradebookMain file to test your implementation.

The main focus of this lab is to learn about:

  • Creating class instances (objects) using constructors.
  • Implementing methods that perform actions (e.g., adding assignments, printing reports).
  • Interacting with objects from a client program (the main method).

Instructions
