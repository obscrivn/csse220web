Coding Exam Instructions

  1. Allowed materials: all labs and homeworks, provided they are in your workspace. You will not be allowed to browse and access any online materials.
  2. Time: 110 minutes
  3. First, go to Moodle and Select Exam 1 Coding part, it will take you to the Exam on Gradescope. Please submit any random file to gradescop submission to get started. This is necessary so that you will be provided a link for the Exam Stater. Once you download Exam starter, close all your browser tabs and contitue with Eclipse.
  4. Download the Exam starter zip file from Gradescope and import it into Eclipse. The exam starter will contain a set of classes and a set of JUnit tests. You will be required to implement the classes to make the JUnit tests pass.
  5. When ready and all tests are passed, you can open now your browser and submit 2 java files to Gradescope
  6. Remember to include your name and reference in javadoc
