GitHub Setup for Your Team

Step 1: Create the repository

  1. Go to GitHub and click New repository
  2. Enter a repository name
  3. You may add a short description
  4. Choose public repo
  5. Initialize it with a README
  6. Add gitignore for Java
  7. Click Create repository

Step 2: Add your teammate as a collaborator

  1. Open the repository on GitHub
  2. Go to Settings → Collaborators or Collaborators and teams
  3. Click Add people or invite a collaborator
  4. Enter your teammate’s GitHub username
  5. Send the invitation
  6. Your teammate must accept the invitation before they can contribute

Step 3: Download GitHub Desktop

  1. Download and install GitHub Desktop
  2. Sign in with your GitHub account

Step 4: Clone the repository

  1. Open your repository on GitHub
  2. Click Code
  3. Choose Open with GitHub Desktop
  4. Select the folder where you want the project saved
  5. Click Clone

Step 5: Push your changes

  1. Make changes in your local project folder.
  2. Open GitHub Desktop and review your changed files.
  3. Enter a short commit message.
  4. Click Commit to main (or your current branch).
  5. Click Push origin to upload your changes to GitHub.

Step 6: Pull before you start working

  1. Before making new changes, open GitHub Desktop.
  2. Click Fetch origin.
  3. If there are new changes on GitHub, click Pull origin.
  4. This helps you stay up to date before you edit code.

Team reminder: Always pull before you start working, and push after you finish a good chunk of work.
